# Connect 4 – Minimax & Alpha-Beta AI

An interactive web-based implementation of the classic Connect 4 game featuring a polished UI, smooth animations, and AI powered by Minimax and Alpha-Beta pruning algorithms. Built entirely with HTML, CSS, and JavaScript, this project demonstrates adversarial search, heuristic evaluation, and game tree optimization in a visually engaging format.

---

## 🎮 Features

- 🧠 **AI Modes**
  - Minimax
  - Alpha-Beta Pruning
- 👥 2 Player Mode (PvP)
- ✨ Glossy 3D Coin Styling
- 🎯 Animated Win-Line Highlight
- 👁 Hover Preview Disc
- 🔁 Smooth Drop Animation
- 🧩 Heuristic Evaluation Function
- 🎛 Mode Toggle (PvP / AI)

---

## 🧠 AI Details

The AI uses:

- **Depth-Limited Minimax**
- **Alpha-Beta Pruning Optimization**
- **Heuristic Evaluation Function**
  - Center column weighting
  - 4-window scoring
  - Threat blocking
  - Offensive prioritization

Alpha-Beta produces identical moves to Minimax but explores fewer nodes, improving efficiency.
